A list of bugs I encounter

Just an unordered list of bugs or unexpected quirks I encounter while at my main desktop.
  • In a jupyter notebook, tqdm's in-place error bar stops being in-place if you interrupt the kernel once while using it.
  • Jupyter notebook "magically" changes over insert mode, overwriting previously-written code.
  • Importing a flat file using SQL Server Management studio, checking "Rich type detection" after already viewing the columns gave an inscrutable error message about ComboBoxDropdown not being valid, and I needed to handle the DataError first. Clicking OK just popped up the same box again. Of coursde it also locked me out of doing anything else. Luckily spamming Escape fixed it.
  • Installing Zen Blue Lite software gives no errors, but then the application won't open and just crash on startup.
  • Clicking "too fast" in Microsoft's Delve for company information deselects a filter box but then doesn't update the results.
  • Using Firefox Focus on mobile, reading a Steve Yegge blog post results in the "got it" option for the cookie banner being hidden underneath the "destroy history" button. You have to do some risky touch gymnastics to click it.
  • In my homegrown notetool, starting the application before syncing the data will result in data loss.
  • There was a floaty black bar on the calendar in microsoft teams, like you see when you can move a window around.
  • Clicking on "Home" while in the Outlook Calendar view from "Meeting" jumped back to meeting after half a second.
  • Links in error messages for Microsoft products tend to be broken. An SSL error message in PowerBI leads me to the "What's New"-page for ADO.NET.
  • Random button saying "Minimize" when opening PyCharm. Clicking it doesn't do anything. Turns out it came from PowerBI!
  • Microsoft Teams room installation doesn't show video or send video. After restarting it does show video, but still doesn't show any. Was unable to resolve.
  • Clicking on a picture in Microsoft teams leads the screen to be empty and unresponsive until I switch over to a different chat, closing the picture.
  • Pinterest doesn't load a shared board in Firefox (with some privacy-friendly settings). Clicking on 'Get Help' redirects after 1 second.
  • Outlook keeps popping up the same "welcome to the new search" message, even though I already have had this version of Outlook for a couple months. Probably desktop-as-a-service related, but still annoying. Also it pops over the browser window sometimes.
  • Extracting a moderately-sized ZIP-archive (60 MB) takes forever and crashes Windows Explorer.
  • Opening Spotify on my phone and clicking on a song does literally nothing until I restart the application.
  • Reddit preview links don't do anything on the old style site.
  • Interrupting a Jupyter notebook kernel mid-import puts it in a broken state. Module numpy has no attribute __config__ when trying to import pandas.
  • Notification dot on Calls tab does not disappear in Microsoft Teams.
  • Whatsapp Web does not show a newline inserted using shift+return until typing the next word.
  • Landing on a Facebook page prevents me from going back to the previous page. Unless it's on Facebook.
  • Scrolling breaks in Jupyter notebooks on output cells which don't have any scrollbar.

Why don't you fix these yourself?

Some software I can't fix because it's closed source. Most software I won't fix because it is too much of an effort to get to know these sprawling code bases merely to fix a few bugs. My own software I will sometimes fix, if I feel like it.
